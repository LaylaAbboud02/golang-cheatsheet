# Golang Cheat Sheet

A Go reference with quick examples for syntax, tooling, standard library patterns, and common workflows.

## Open Locally

Open `index.html` in your browser.

## Project Structure

- `index.html` - page shell and controls
- `styles.css` - visual styling
- `script.js` - small app entry point
- `src/app.js` - app rendering, search/filtering, navigation, notes, study packs, and snippet buttons
- `src/study-pack.js` - Markdown study pack export behavior
- `src/data/sections.js` - cheat sheet content
- `src/data/metadata.js` - difficulty levels and runnable Playground examples
- `src/data/notes.js` - curated field notes, idioms, and gotchas
- `src/data/ui.js` - UI copy, About content, and mascot messages
- `src/utils.js` - shared helper functions
